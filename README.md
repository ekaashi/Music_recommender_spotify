# 🎵 Music Recommender System

A machine learning–based music recommendation system that suggests songs similar to a selected track using **TF-IDF vectorization** and **cosine similarity**. The application is built with **Python** and **Streamlit**, and it fetches album cover images using the **Spotify API** to provide an interactive user experience.

## 🚀 Features

* Content-based music recommendation
* NLP text preprocessing (tokenization, stopword removal, stemming)
* TF-IDF vectorization for feature extraction
* Cosine similarity for finding similar songs
* Interactive UI built with Streamlit
* Album cover images fetched from Spotify API

## 🛠 Technologies Used

* Python
* Pandas
* Scikit-learn
* NLTK
* Streamlit
* Spotipy (Spotify API)

## 📌 Project Goal

This project demonstrates how **machine learning and NLP techniques** can be used to build an intelligent recommendation system that suggests relevant music to users.
